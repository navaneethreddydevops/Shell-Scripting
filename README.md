# Shell-Scripting

Repo for shell syntax and Code files for Reference

# Shortcuts in shell

```
touch ~/.bashrc

** alias newcommand='yourcommand -arguments' **

alias c='clear'
alias cp="cp -i"
alias ls="ls -al"
alias di="docker images"
alias dc="docker ps"
alias gp="git push origin master"
alias gf="git fetch; git pull"
```

# Unalias (Removing the Alias)
```
unalias c
unalias dc
```
